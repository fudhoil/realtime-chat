# Realtime Chat

This is a realtime chat application built with Next JS 13, Upstash, and Pusher.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
